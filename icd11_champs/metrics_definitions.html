CHAMPS vs WHO ICD-11 Feature Concordance Definitions

These tables compare DeCoDe underlying-cause assignments with official WHO ICD-11 cause-profile features for the 77 target causes. They are aggregate review signals, not case adjudications and not proof that a DeCoDe assignment is correct or incorrect.

The main denominator is cause-specific: for a cause page, percentages use cases whose DeCoDe underlying cause is that ICD-11 cause. Age tabs restrict the denominator to the selected site age group.

The older profile-match metrics are still saved locally as audit outputs, but the main dashboard now emphasizes feature prevalence because it is easier to interpret and closer to the CHAMPS-vs-WHO comparison question.

MetricValue ShownDefinitionHow To Read It
WHO Feature ConcordancePercent of assigned casesFor one ICD-11 cause, each row is a WHO ICD-11 feature. The percent is the share of CHAMPS cases assigned to that cause as the DeCoDe underlying cause that contain the feature in allowed CHAMPS evidence fields.This is the clearest within-cause comparison: it says how often CHAMPS cases assigned to a cause show each WHO feature for that cause.
Source ScopeCause-specific, parent/block fallback, or sparseA label showing whether the WHO feature came from the cause-specific ICD-11 entity fields, from broader WHO parent/block context, or from an explicit source-sparse placeholder.Parent/block fallback rows are useful context when entity text is sparse, but they should not be read as cause-specific diagnostic criteria.
Additional CHAMPS FeaturesPercent of assigned casesFeatures detected in allowed CHAMPS evidence fields for cases assigned to a cause, where the feature is not part of that assigned cause's WHO profile but is represented somewhere else in the 77-cause WHO feature vocabulary.These rows highlight what CHAMPS is seeing beyond the WHO representation for the assigned cause. They can reflect true extra pathology, comorbidity, broad WHO text, or deterministic dictionary limitations.
Numeric Boundary MatrixBelow / within / above / not assessableFor WHO age, gestational-age, or birthweight boundaries, the table shows the share of assigned CHAMPS cases below, within, above, or missing the required numeric measurement.This replaces the older single boundary-conflict rate for numeric boundaries. Nonnumeric boundaries are flagged but not forced into a numeric matrix.
Primary Cause DenominatorN casesFor a cause page, N is the number of CHAMPS cases whose DeCoDe underlying cause is that ICD-11 cause. Age tabs restrict N to cases in the selected site age group.Cause-specific percentages use this cause-specific N, not the overall 8,304 target-case denominator.
Legacy Profile-Match MetricsLocal audit outputsThe previous exact-code, hierarchy, rank, specificity, and family-confusion outputs are still written to reproducible CSV/JSON files under the metrics outputs directory.They are no longer the main dashboard because they are harder to interpret than within-cause feature prevalence.
